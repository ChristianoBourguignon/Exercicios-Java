# 🧪 Java Playground - Exercícios

Este repositório é meu **espaço de testes e prática com Java**. Aqui você vai encontrar vários exercícios, exemplos e pequenos projetos feitos enquanto estudo e exploro a linguagem.

## 📚 Sobre

A ideia é usar essa pasta como um **playground pessoal**:  
Nada aqui precisa estar perfeito — o objetivo é praticar, errar, aprender e evoluir. 💡

## 🛠 Conteúdo

- Exercícios de lógica
- Prática com POO (Programação Orientada a Objetos)
- Testes com classes, métodos e estruturas de dados
- Projetos simples para fixar o aprendizado

## 🚧 Aviso

Este repositório **não tem um padrão fixo**, estrutura definida ou código pronto pra produção.  
É um espaço livre de julgamentos, feito só para experimentar e me desenvolver como dev 💻✨

## ☕ Bora codar?

Se quiser trocar uma ideia sobre Java ou dar sugestões, é só abrir uma issue ou me chamar!
